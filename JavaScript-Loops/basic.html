Loops

Loops iterate through a set function and can execute a block of code a given number of times.[1] There are several different kinds of Loops, the most common are for and while loops. The key differences between these two, is that a for loop will only iterate for a certain period of time or number of loops. A while loop however, will iterate through the loop so long as a scified condition is true.
